# python-web-tornado-api-postgres-pop

## Description
Simple api for a tornado project.
Creates and return table `pop`.

## Tech stack
- python
- tornado
- sqlalchemy
- postgres

## Docker stack
- python
- postgresql:alpine

## To run
`sudo ./install.sh -u`
- http://localhost/pop

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
